# Zenith Editor v0.1.0

First public macOS release of Zenith Editor.

## What's Included

- Local subtitle generation with Whisper model variants
- Subtitle styling controls (font, size, color, background, vertical position)
- Canvas-burn subtitle rendering for preview/export parity
- Timeline editing workflow with media bin, preview, inspector, and export

## Notes

- Requires `ffmpeg` and `ffprobe` installed on macOS.
- Install command: `brew install ffmpeg`
